Bitovi Components

A collection of reusable components for CanJS that can be loaded by:

  • StealJS + ES6
  • npm / Browserify / CJS
  • RequireJS / AMD
  • Standalone

Autocomplete

An auto-suggest/auto-complete widget

Site | Code

C3

Build C3 charts with CanJS components

Site | Code 
npm install bit-c3 --save
